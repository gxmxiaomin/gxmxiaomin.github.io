python介绍

发表于2019-05-26|更新于2022-06-01|python
|阅读量:

1.计算机的组成
由硬件和软件两部分组成(写代码主要时实现软件层面的功能)
3.Python语言介绍
Python是一门解释型的编程语言
1991年,第一个Python编译器诞生,是由吉多本人开发
Python是由ABC发展起来的,并结合了unix shell和C语言的习惯
优点:简单,易学,免费,开源,高层语言,可移植性,面向对象,可扩展性,规范的代码,解释型语言,丰富的库.
缺点:唯一的缺点就是执行起来效率慢,这个是解释型语言所通有的.
Python的设计哲学,优雅,明确,简单.
应用场景:web开发,操作系统管理,服务器,运维的自动化脚本,网络爬虫,科学计算,桌面软件,服务器软件()网络软件),游戏.
交互式编程 REPL分别表示为Read读,Eval执行,Print打印,Loop循环.
交互式编程的优缺点
优点:适合学习/验证Python语法或者局部代码
缺点:代码不能保存,不适合运行太大的程序.
注释的分类:单行注释以#开头#右边的东西当做说明起辅助说明作用.
多行注释 以’’’开头 , 以’’’结束
变量及数据类型
变量的定义:对于重复使用,并且经常需要修改的数据,可以定义为变量,来提高编程效率. 变量名=变量值.
变量的类别:数字(Numbers):int:整形 long:长整型 float:浮点型 complex:复数
字符串(string)
列表(list)
元祖(tuple)
字典(dictionary)
查看数据类型:数据才有类型,变量没有类型(如果临时想要查看一个变量存储的数据类型,可以使用type(变量的名字))
标识符和关键字1.标识符命名规则:由字母,下划线和数字组成数字不能开头 严格区分大小写 不能使用关键字
2.命名规范:标识符命名要做到顾名思义 遵守一定的命名规范
小驼峰命名法 第一个单词首字母小写,第二个字母首字母大写
大驼峰命名法: 每个单词的首字母都是大写字母.
还有一种命名方法是用下划线”_”来连接所有单词.
输出语句 格式化输出(在程序中看到了%这样的操作符),这就是Python中的格式化输出
常用格式符 %% 输出%号 %S 字符串 %d 有符号十进制整数 %f 浮点数 %c字符 %u无符号十进制整数 %o 八进制整数
%x或%X 十六进制整数小写字母0x(大写字母0X) %e或%E 科学计数法小写e(大写E) %g或%G %e和%f的简写(%E和%F的简写)
换行输出:在输出的时候,如果有\n那么此时\n后面的内容会在另一行显示.
输入语句 input() 注意:input()的小括号中放入的是提示信息,用来在获取数据之前给用户一个简单提示
input()在从键盘获取了数据以后,会存放到等号右边的变量中.
input()会把用户输入的任何值都做为字符串来对待.
