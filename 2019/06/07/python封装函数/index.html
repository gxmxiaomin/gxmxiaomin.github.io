python封装函数

发表于2019-06-07|更新于2022-06-07|python
|阅读量:

什么是函数
函数的定义:如果在开发程序时,需要某快代码多次执行.为了提高编写的效率以及更好的维护代码,
需要把具有独立功能的代码块组织位一个小模块,这就是函数.
函数的基本使用
函数的特点是:定义时函数体内的代码不会执行,调用函数时,函数体内的代码才会执行.
每次调用函数时,函数都会从头开始执行,当这个函数中的代码执行完毕后,意味着调用结束了
当然如果函数中执行到了return也会结束函数.
{def 函数名(形参):
封装的代码
调用函数: 函数名(实参)}
函数的参数(入口)
形参:放在函数定义时,形式上的参数,具有实际的意义但是没有实际的值,
当函数被调用时才会进行赋值,用于接收参数用的.(等待赋值的变量名)
实参:放在函数调用时 用于给形参进行赋值,用来传递函数用的.(真正要进行运算的数据)
函数的返回值(出口):
return 一个函数中可以有多个return 关键字 有终止代码的作用
通过return返回多个值 格式: return 值1,值2,……..多个值之间使用英文逗号分隔,整体返回一个元组.
所谓”返回值”,就是程序中函数完成一件事后,最后给调用者的结果
使用返回值的前提需求就是函数调用者想要在函数外使用计算结果.
函数的嵌套调用:
一个函数里面又调用了另外一个函数,这就是所谓的函数嵌套调用
如果函数A中,调用了另外一个函数B,那么先把函数B中的任务都执行完毕之后才会回到上次执行函数A执行的位置.
函数高级:
局部变量和全局变量
局部变量,就是在函数内部定义的变量
其作用范围就是函数内部,即只能在这个函数内部使用,不能在函数外部使用.
因为其作用范围只是在自己的函数内部,所以不同的函数可以定义相同的局部变量
局部变量的作用,为了临时保存数据需要在函数中定义变量来进行存储.
当函数调用时,局部变量被创建,当函数调用完成后这个变量就不能够是用了
全局变量:如果一个变量,既能在一个函数中使用,也能在其他函数中使用,这样的变量就是全局变量.
在函数外边定义的变量叫做全局变量
全局变量能够在所有的函数中进行访问.
当函数时出现局部变量和全局变量相同名字时,函数内部中的 变量名=数据 此时理解为定义了一个局部变量,而不是修改全局变量的值.
修改全局变量:
如果在函数中出现global 全局变量的名字,那么这个函数中出现和全局变量名相同的 变量名=数据 也理解为对全局变量进行修改,而不是定义局部变量.
如果在一个函数中需要对多个全局变量进行修改,那么可以一次性全部声明,也可以分开声明.
查看所有的全局变量和局部变量
Python提供了两个内置函数globals()和ocals()可以用来查看所有的全局变量和局部变量.
函数返回值详解
一个函数中不可以用多个return,因为return有结束函数的作用.
想要返回多个值,可以在return后面加多个数据,可以是元祖,列表,字典等只要是能够存储多个数据的类型,就可以一次性返回多个数据.
如果return后面有多个数据,默认是元祖.
对返回的数据直接拆包
拆包时注意,需要拆的数据的个数和变量的个数相同,否则程序会异常
除了对元祖拆包之外,还可以对列表,字典等拆包.
函数参数详解
1.缺省参数
调用函数时,缺省参数的值如果没有传入,则取默认值.
在形参中默认的值的参数,称之为缺省参数
注意:带有默认指的参数一定要位于参数列表的最后面.
2.不定长参数
有时可能需要一个函数能处理比当初声明时,更多的参数,这些参数叫做不定长参数,声明时不会命名.
加了 星号()的变量args会存放所有未命名的变量参数,args为元祖
而加**的变量kwargs会存放命名参数,即形如key=value 的参数,kwargs为字典
3. 缺省参数放在args后面
如果很多个值都是不定参数,那么这种情况下,可以将缺省参数放到args后面,但是如果有kwargs的话,kwargs必须放到最后的.
